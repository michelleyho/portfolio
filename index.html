System Design Steps

Basic Building Blocks

Basic guideline on how to approach a System Design

01

Clarify Functional Requirements

Identify and ask clarifying questions of the system design problem. Ask for functional requirements

02

Clarify - NonFunctional Requirements

Identify and ask clarifying questions of the system design problem. Ask for non functional requirements

03

Identify Basic Building Blocks

Mention basic components that you will need to build various parts of the system

04

Scale Design

Identify ways to scale the design

05

API Design

High level overview of the API design for necessary functions

06

Summarize

Summarize on your design on how it meets both the functional and non functional requirements
